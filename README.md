# Lunari Privacy Policy

This Privacy Policy describes how our Discord bot collects, uses,
and stores data.

## 1. Data We Collect
We collect and store only the minimum data required to provide
bot functionality. This may include:
- Discord user IDs
- Discord guild IDs
- Feature-related data such as AFK status, profile information,
  and relationship (marry) data

## 2. Data We Do Not Collect
We do not store:
- Message content
- Private messages (DMs)
- Personal information outside of Discord identifiers
